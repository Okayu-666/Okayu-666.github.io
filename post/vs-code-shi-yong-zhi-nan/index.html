VS Code使用指南

2023-02-24

1.好用配件

  1. Chinese：中文界面
  2. Prettier：代码美化
  3. Wisen-Translate：翻译插件
  4. Live preview：实时预览html
  5. Search/Translate Hero：搜索跳转和翻译
  6. vscoed-icons：美化图标

2.VS Code配置

  1. Format On Save：勾选，代码保存后自动格式化
  2. Format On Paste：勾选，粘贴时自动格式化
  3. Default Formatter：选择Prettier，默认格式化插件
  4. Linked Editing：勾选，同时重命名html标签名

3.常用快捷键

  1. ctrl+D：选择下一个相同字符
  2. Alt+鼠标左键：增加光标
  3. alt+shift+上下键：快速复制内容至上/下行
  4. crtl+F：查找内容
