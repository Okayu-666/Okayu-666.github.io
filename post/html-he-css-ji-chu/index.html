1.HTML

1.基础标签

<!DOCTYPE html> 
<html> 
<head> 
<meta charset="utf-8"> 
<title>菜鸟教程(runoob.com)</title>
</head>
<body> 
<h1>我的第一个标题</h1> 
<p>我的第一个段落。</p> </body> 
</html>

![[Pasted image 20230219231715.png]]

  •  声明为 HTML5 文档
  • < html >< /html> 元素是 HTML 页面的根元素
  • < head> < /head>元素包含了文档的元（meta）数据，如  定义网页编码格式为 utf-8。
  • < title> < /title>元素描述了文档的标题
  • < body>< /body> 元素包含了可见的页面内容
  • < h1> < /h1>元素定义一个大标题，总共有六级标题，在h后改数字可换
  • < p> < /p>元素定义一个段落
  • < i>< /i> 定义一段斜字体文字
  • < hr> 定义一条分割线
  • < b>< /b> 标签内字体加粗，不变大
  • < br> 换行标记
  • &npsp; 加空格
